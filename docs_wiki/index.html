Wiki do produto

De conversa a specs — sem virar caos

O Create Specs Setup configura, uma vez, o repositório de especificações do seu produto. Depois, o time especifica features, bugs e documentação conversando — com artefatos rastreáveis.

Bem-vindo

Esta wiki é para quem usa ou apresenta o Create Specs Setup: líderes, POs, devs e quem precisa entender o valor sem abrir código-fonte.

Em uma frase

É o “create-react-app” das especificações: você roda o setup uma vez e nasce o {projeto}-specs pronto para o dia a dia.
