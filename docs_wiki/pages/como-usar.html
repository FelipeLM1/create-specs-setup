Como usar esta wiki

Um mapa rápido para encontrar respostas sobre o Create Specs Setup — pensado para leitura agradável, não para manual técnico.

Para quem é

Esta wiki serve a quem precisa entender ou apresentar o Create Specs Setup: cliente, líder, PO e dev — todos na mesma linguagem de produto.

O que você não encontra aqui

Código-fonte, contratos de API ou detalhes de implementação. Isso fica no README técnico e nos templates do repositório.

O que encontrar

As páginas seguem um fio narrativo — e, quando ajuda, usam diagramas:

  1. O problema — qual dor o produto ataca.
  2. O que resolve — o valor em linguagem de negócio.
  3. Como funciona — o comportamento esperado no processo (às vezes com fluxo visual).
  4. Como fazer — o passo a passo para começar e seguir.

Como ler

Use o menu superior para navegar. Nas páginas longas, a barra lateral destaca a seção em que você está.

Animações leves acompanham a leitura (e respeitam se o seu sistema pediu menos movimento).

Tema deste preview

Esta pasta docs_wiki/ no repositório create-specs-setup é a wiki do próprio produto, também usada para validar o visual gerado pelos templates.

  • Tema: suave
  • Cores: paleta do tema (sem override de marca)
  • Escrita: narrativo
  • Animação: sutil

Para o time

Para experimentar outro visual, troque o CSS do tema nos <link> e o atributo data-motion / data-theme no HTML.
