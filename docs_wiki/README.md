# Docs wiki — Create Specs Setup

Wiki HTML de produto deste repositório (também serve de preview dos templates).

## Abrir

Abra [`index.html`](index.html) no navegador.

## Configuração deste preview

| Campo | Valor padrão |
|-------|--------|
| Tema | `suave` (trocável no botão ⚙️) |
| Cores da marca | não |
| Escrita | `narrativo` |
| Animação | `sutil` (trocável no botão ⚙️) |

No canto inferior direito há um **botão de config** para alternar tema e animação (salvo no navegador). Só existe neste preview do create-specs-setup — não vai para as instâncias geradas.

Templates-fonte: `templates/instance/docs_wiki/`.
